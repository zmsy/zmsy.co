About Me

This site is created and maintained by me, Zach Morrissey. I created it from scratch using a number of different technologies:

  • Python - Visual analysis, data prep, pretty much anything actually.
  • Lektor - For content management. Soon to be for static site generation.
  • Bulma - Excellent CSS-only front-end framework that drastically simplifies structuring a website.
  • HTML/CSS/JS - The basics.
  • Webpack - Bundling all of my resources and being able to define frontend dependencies easily.
  • Let's Encrypt - So I can serve this website over HTTPS (still working on it).

I live in Seattle, WA with my girlfriend and my dog Howie (pictured to the right). Interests include good beer, photography, music and music technology, guitars, tinkering with electronics, stats, data visualization, football, baseball, videogames, and computers.

This site was built with the purpose of becoming more familiar with the processes involved in building and deploying a website, organizing my app code to correctly serve multiple request endpoints, and most importantly for shameless self-promotion.

All photos on the site are taken by me, with the exception of ones clearly labeled with their sources below them. All SVG, data visualizations, etc, are created by me.
